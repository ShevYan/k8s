# 无障碍上网

## 原理

Socks和Http都是代理协议，都只是针对浏览器和http访问的，跟vpn有本质的差异，vpn是从2-3层打通网络，所以是能够ping通的，而socks和http不会影响ping。因此，一般软件都需要设置代理，通常是http_proxy和https_proxy环境变量。比如git就可以配置代理。
